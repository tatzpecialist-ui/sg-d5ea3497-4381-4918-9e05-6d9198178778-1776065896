# 13 Media Works

## Vision
Professional media production booking platform for a digital nomad videographer/editor. Target clients: businesses seeking video editing, live event coverage, documentary production, and corporate AV presentations.

## Design
- **Primary**: 0 0% 12% (deep charcoal #1E1E2E)
- **Background**: 240 6% 10% (dark slate)
- **Foreground**: 60 30% 96% (warm white)
- **Accent**: 40 48% 58% (warm amber #C9A961)
- **Muted**: 240 4% 16% (muted slate)
- **Card**: 240 5% 14% (card surface)
- Headings: Playfair Display (serif)
- Body: Raleway (sans-serif)
- Style: Cinematic luxury, ample negative space, high contrast typography

## Features
- Hero landing with service overview
- Interactive booking/scheduling system with Supabase
- Portfolio showcase (video editing, events, documentary, corporate)
- Service detail pages
- Client intake forms
- Admin dashboard for booking management
